# E-Commerce Frontend

Vue 3 + TypeScript + Vite + Pinia + TailwindCSS + Bun

## Prerequisites

Install Bun (if not already installed):
```bash
# Windows (PowerShell)
irm bun.sh/install.ps1 | iex

# macOS/Linux
curl -fsSL https://bun.sh/install | bash
```

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
# or simply
bun dev
```

## Build

```bash
bun run build
# or simply
bun build
```

## Preview

```bash
bun run preview
# or simply
bun preview
```
